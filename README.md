# Sakura Mahar - Platform Pernikahan Modern

Sakura Mahar adalah platform pernikahan modern yang dirancang untuk memudahkan Anda merayakan momen spesial dengan sempurna. Aplikasi ini dibangun dengan React dan menyediakan berbagai fitur lengkap untuk mengelola pernikahan Anda.

## Fitur Utama

- **Mahar Digital** - Terima mahar secara digital dengan aman dan terpercaya
- **Undangan Digital** - Buat dan kirim undangan pernikahan yang elegan
- **Manajemen Tamu** - Kelola daftar tamu, RSVP, dan tempat duduk
- **Jadwal Lengkap** - Buat jadwal acara pernikahan yang terorganisir
- **Galeri Momen** - Bagikan foto dan video pernikahan Anda
- **Keamanan Terjamin** - Data dilindungi dengan enkripsi tingkat bank

## Persyaratan

- Node.js (v14 atau lebih tinggi)
- npm atau yarn

## Instalasi

1. Clone repository atau extract file ke folder Anda:
```bash
cd sakura-mahar
```

2. Install dependencies:
```bash
npm install
```

3. Jalankan aplikasi dalam mode development:
```bash
npm start
```

4. Buka browser Anda dan akses `http://localhost:3000`

## Build untuk Production

Untuk membuat build production:
```bash
npm run build
```

File production akan berada di folder `build/`
