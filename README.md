# Sakura Mahar

Sakura Mahar adalah aplikasi web untuk katalog produk mahar, checkout, login user, dan pengelolaan admin.

## Fitur

- Landing page produk mahar
- Login, daftar, reset password, dan ganti password
- Checkout produk
- Panel admin
- Penyimpanan data user/auth ke Firestore

## Teknologi

- React
- Express
- Vercel
- Firebase Firestore

## Menjalankan Project

```bash
npm install
npm start
```

App berjalan di `http://localhost:3000`.

## Build

```bash
npm run build
```

## Catatan

- Frontend dan backend production berjalan di Vercel.
- Data user/auth production disimpan permanen di Firestore.
