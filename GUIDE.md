# Panduan Menggunakan Sakura Mahar

## Memulai

1. **Daftar Akun**
   - Klik tombol "Daftar" di header
   - Isi data diri Anda
   - Verifikasi email Anda

2. **Buat Acara Pernikahan**
   - Masuk ke dashboard
   - Klik "Buat Acara Baru"
   - Isi detail pernikahan Anda

3. **Buat Undangan Digital**
   - Pilih template undangan
   - Sesuaikan dengan tema Anda
   - Bagikan link ke tamu

## Fitur Utama

### Mahar Digital
- Tentukan jumlah mahar
- Pilih metode pembayaran
- Lacak pembayaran masuk secara real-time

### Manajemen Tamu
- Tambahkan daftar tamu
- Kelola RSVP otomatis
- Atur tempat duduk

### Undangan Digital
- Pilih dari berbagai template
- Tambahkan foto Anda
- Kirim via WhatsApp, Email, atau Link

### Galeri Foto
- Upload foto pernikahan Anda
- Bagikan dengan keluarga dan teman
- Keamanan password terjamin

## Tips Berguna

- Mulai persiapan minimal 3 bulan sebelum pernikahan
- Update status RSVP tamu secara berkala
- Backup data penting Anda secara teratur
- Hubungi support jika ada masalah

## FAQ

**Q: Apakah data saya aman?**
A: Ya, semua data dilindungi dengan enkripsi SSL dan backup harian.

**Q: Berapa lama undangan digital bisa diakses?**
A: Undangan bisa diakses selamanya selama akun Anda aktif.

**Q: Bagaimana cara mengunduh undangan?**
A: Anda bisa mengunduh sebagai PDF atau gambar melalui menu download.

**Q: Apakah ada batas jumlah tamu?**
A: Tergantung paket. Paket Platinum memiliki tamu unlimited.

**Q: Bagaimana cara menghubungi support?**
A: Gunakan live chat di aplikasi atau email ke support@sakuramahar.com
